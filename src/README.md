## Quick Start

1. Place your image in the `input/` folder
2. Run: `python src/blur_tool.py input/your_image.jpg`
3. Draw outline around person/object
4. Press 's' to save, 'r' to reset, 'q' to quit

## Controls
- **Mouse**: Click and drag to draw outline
- **S**: Save current result
- **R**: Reset and start over
- **Q**: Quit application
